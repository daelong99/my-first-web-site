HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications. web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperance of the document.

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive froms, may be embedded into the rendered page. It provides a means to create structural semantics for text to create structured documents by denothing structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineater by tags, written using angle brackets.
